INTRODUCTION
------------
This module provides an Omani Rial currency support for Drupal commerce.

REQUIREMENTS
------------
This module requires the following modules:
  * Commerce (https://drupal.org/project/commerce)

INSTALLATION
------------
  * Enable the Commerce Omani Rial module (https://www.drupal.org/documentation/install/modules-themes/modules-7).
  * Clear cache.

CONFIGURATION
-------------
Configuration for commerce currencies can be found at the following URL:

admin/commerce/config/currency

From this page simply expand the 'Enabled Currencies' drop down and you will
see 'Omani Rial' appear in the list.

To enable 'Omani Rial' just tick the box next to it in the list and click the
'Save configuration' button.
